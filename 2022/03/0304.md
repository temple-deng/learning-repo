## conic-gradient()锥形渐变简介

> 摘自 https://www.zhangxinxu.com/wordpress/2020/04/css-conic-gradient/

语法：   

```css
conic-gradient( [ from <angle> ]? [ at <position> ], <angular-color-stop-list> )
```    

由3部分组成：   

1. 起始角度
2. 中心位置
3. 角渐变断点

```css
.example {
    width: 300px;
    height: 150px;
    background-image: conic-gradient(white, deepskyblue);
}
```
